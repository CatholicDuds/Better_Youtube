# Diário de aprendizado do Clarity

Esta pasta é a memória de longo prazo para estudar com o Codex. Depois de cada bloco de dois vídeos e uma leitura, o Clarity baixa um arquivo `clarity-diario-AAAA-MM-DD.md`. Mova esse arquivo para `entradas/`.

## Como usar com o Codex

Peça, por exemplo:

> Leia minhas entradas dos últimos 7 dias. Identifique os princípios recorrentes, lacunas de compreensão e contradições. Depois prepare uma aula socrática de 30 minutos, começando pelos fundamentos e avançando apenas quando minhas respostas demonstrarem domínio.

O Codex pode comparar dias, criar revisões espaçadas, gerar perguntas e montar uma trilha de estudo. Os textos continuam sendo seus; não os envie a serviços externos sem intenção explícita.

## Estrutura

- `entradas/`: arquivos diários exportados pelo Clarity.
- `MODELO-DE-AULA.md`: roteiro para transformar o diário em uma aula progressiva.
